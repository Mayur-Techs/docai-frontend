Architecture

Rules first. AI only when needed.

Most document AI tools send every PDF directly to an LLM and pay for every token. We don't. Structured PDFs are handled entirely by code — AI only fills the gaps.

STEP 01
📥

PDF Upload

Drag and drop or upload via API. Supports any Indian invoice format — GST, commercial, freight, purchase orders.

Instant
STEP 02
📊

Table + Text Extraction

pdfplumber reads all tables (line items) and text. PyMuPDF as fallback for unusual PDFs. Zero API cost.

Rules · Free
STEP 03
🔍

Regex Pattern Match

GSTIN (15-char pattern), invoice numbers, dates, amounts, totals — extracted by regex in under 100ms. If confidence ≥ 70%, AI is skipped entirely.

Rules · Free
STEP 04
🤖

AI Gap Filling

Only if rules confidence is below 70%, Gemini Flash fills the missing fields specifically — not the whole document. Avg: 150 tokens, not 2,000.
